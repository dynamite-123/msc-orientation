## 1. Even or Odd
Write a program that takes an integer as input and prints `"Even"` if the number is even, and `"Odd"` if the number is odd.

### Sample Input
```
7
```

### Sample Output
```
Odd
```

### Sample Input
```
10
```

### Sample Output
```
Even
```
